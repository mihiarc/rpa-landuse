# 📊 **RPA Land Use Streamlit App**

The Streamlit app has **7 tabs**:

#### **Tab 1: Overview** 
- RPA Assessment methodology and key findings
- Scenario descriptions (5 key climate/economic scenarios)
- Data processing information and technical details

#### **Tab 2: Data Explorer**
- Interactive exploration of all datasets
- Column information and data previews
- **Download functionality** for CSV export
- Basic statistics for each dataset

#### **Tab 3: Land Use Flow Diagrams**
- **Sankey diagrams** showing land use transitions
- Interactive scenario and time period selection
- Visual representation of land conversion patterns

#### **Tab 4: Urbanization Trends**
- **Core feature**: Answers "Where is urban development rate highest?"
- County and state-level rankings
- Interactive charts showing transitions to urban land
- Time series analysis of forest, cropland, and pasture conversion
- **Downloadable results** with detailed metrics

#### **Tab 5: Forest Transitions**
- Forest land conversion patterns by destination
- Counties experiencing highest forest land loss
- Detailed RPA Assessment findings integration

#### **Tab 6: Agricultural Transitions**
- **Core feature**: Answers "Where is agricultural land loss rate highest?"
- Analysis of both cropland and pasture transitions
- Source/destination filtering capabilities
- **Top counties by agricultural loss rate** with downloads
- National trends visualization

#### **Tab 7: State Map**
- **Geographic visualization** using Folium Interactive Map
- Choropleth maps showing land use changes
- State-level aggregation of county data

#### **Interactive Analysis Tools**
- **Dynamic filtering** by scenario, time period, geographic level
- **Real-time calculations** of transition rates and metrics

#### **Download Functionality**
- **CSV export** with complete metadata
- **FIPS codes** included for GIS integration
- **Top N results** and full dataset options

### 🎯 **Key Policy Questions Answered**

App specifically addresses critical policy questions:

1. **"Where is urban development rate highest?"**
   - County-level rankings with detailed metrics
   - Regional comparisons across scenarios
   - Downloadable datasets for further analysis

2. **"Which areas are losing the most forest land?"**
   - Forest conversion hotspots by county/state
   - Analysis by destination land use
   - Regional and temporal patterns

3. **"Where is agricultural land loss rate highest?"**
   - Agricultural conversion hotspots
   - Source analysis (cropland vs. pasture)
   - Destination analysis (converted to what)

4. **"How do different scenarios affect land use patterns?"**
   - Scenario comparison tools
   - Climate vs. socioeconomic impact analysis

#### **Scenarios Covered**
- **4 Integrated RPA scenarios** for policy analysis
- **Climate projections**: RCP4.5 (lower warming) vs RCP8.5 (higher warming)
- **Socioeconomic pathways**: SSP1-5 (different growth patterns)
- **Ensemble projections**: Average across all scenarios

#### **Geographic Coverage**
- **County-level data** for entire conterminous United States
- **State and regional aggregations**
- **FIPS codes** for GIS integration

#### **Temporal Coverage**
- **2020-2070 projections** in decade intervals
- **Historical baseline** (2020)
- **5 time periods**: 2020-2030, 2030-2040, 2040-2050, 2050-2060, 2060-2070

### 🎯 **Use Cases & Impact**

Application serves multiple stakeholder groups:

- **Policy Makers**: Evidence-based land use planning decisions
- **Researchers**: Academic analysis of land use change patterns  
- **Land Managers**: Regional planning and resource allocation
- **Environmental Groups**: Conservation priority identification
- **Urban Planners**: Development impact assessment
